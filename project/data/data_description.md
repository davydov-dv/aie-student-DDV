### Описание датасета

Датасет UNSW-NB15 создан на основе сырого сетевого трафика, сгенерированного инструментом **IXIA PerfectStorm** в лаборатории Cyber Range Австралийского центра кибербезопасности. С помощью утилиты **Tcpdump** было перехвачено 100 ГБ необработанного трафика (PCAP-файлы), который затем был преобразован в признаки инструментами **Argus** и **Bro-IDS**.

Датасет содержит **9 типов сетевых атак** и нормальный трафик:

| № | Тип атаки | Описание |
|---|-----------|----------|
| 1 | **Fuzzers** | Атаки, связанные с переполнением буфера |
| 2 | **Analysis** | Анализ уязвимостей и сканирование |
| 3 | **Backdoors** | Использование бэкдоров |
| 4 | **DoS** | DDoS-атаки (отказ в обслуживании) |
| 5 | **Exploits** | Эксплойты уязвимостей |
| 6 | **Generic** | Универсальные атаки |
| 7 | **Reconnaissance** | Разведка (сканирование портов, хостов) |
| 8 | **Shellcode** | Выполнение вредоносного кода |
| 9 | **Worms** | Сетевые черви |

## Описание признаков (42 колонки)

Каждая запись в датасете представляет собой **сетевой поток** (flow) — агрегированную информацию о сетевом соединении.

### Основные признаки

| Признак | Тип | Описание |
|---------|-----|----------|
| `id` | int | Идентификатор записи |
| `dur` | float | Длительность соединения (секунды) |
| `proto` | str | Протокол (tcp, udp, icmp и т.д.) |
| `service` | str | Сервис (http, dns, ftp, ssh и т.д.) |
| `state` | str | Состояние соединения (FIN, INT, CON и т.д.) |
